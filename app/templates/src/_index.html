Welcome to Jeti.

Jeti is a simple build system to create fast and optimized static sites which are blog aware. It leverages all the great features of Jekyll and Foundation 6 and wraps it in handy Yeoman Generator.

To find out more check out the documentation for Foundation and Jekyll below, or have a look over the demo blog posts on this site.

Feel free to report bugs, issues and PRs to the Github page
